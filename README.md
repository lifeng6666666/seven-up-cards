# 《人生七年》人物卡片与成长轨迹

一个基于纪录片《人生七年》（Up Series）的交互式数据可视化网站，用于探索14位参与者跨越56年的人生轨迹。

## 项目简介

《人生七年》是英国导演迈克尔·艾普特从1964年开始拍摄的纪录片系列，追踪14位来自不同社会阶层的儿童，每隔7年记录一次他们的生活状态，至今已拍摄至《63 Up》。本项目将纪录片中的人物数据结构化，通过交互式卡片和可视化图表，帮助用户探索社会阶层、教育、婚姻、职业等因素与人生幸福感的关系。

## 网站功能

### 1. 人物卡片筛选与展示

- **多维度筛选**：支持按姓名、年龄节点、婚姻状况、教育程度、生活满意度等字段组合筛选
- **卡片展示**：以卡片形式展示人物基本信息，包括：
  - 中英文姓名
  - 成长背景（上层/精英教育、中产、工人阶级、福利院成长、乡村）
  - 当前年龄节点的生活状态快照
  - 人物简介
  - 标签（如"east-london"、"taxi-driver"、"jockey"）

### 2. 成长轨迹探索

点击人物卡片后，侧边栏展示详细的人生轨迹：

- **时间轴视图**：展示7岁至63岁各年龄节点的关键信息
- **数值型指标折线图**：
  - 生活满意度评分（0-5分）
  - 子女数量变化
- **类别型指标时间轴**：
  - 婚姻状况（单身、已婚、离异、再婚、丧偶等）
  - 教育阶段（小学、中学、大学、研究生等）
  - 工作类别（学生、学者、司机、教师、家庭照护等）
  - 居住地变化

### 3. 对比分析功能

支持按不同维度分组对比：

- **分组条件**：
  - 婚姻状况
  - 教育程度
  - 成长背景（社会阶层）
  - 工作类别
  - 居住地

- **对比指标**：
  - 平均生活满意度
  - 平均子女数量
  - 人数统计

- **年龄节点选择**：可查看特定年龄段的分组对比结果

## 可探索的分析问题

### 1. 教育与生活幸福度的关系

**问题**：不同教育程度的人群，生活满意度是否存在显著差异？

**探索方法**：
- 在对比分析中选择"教育"作为分组条件
- 选择"平均生活满意度"作为对比指标
- 切换不同年龄节点（21岁、35岁、49岁、63岁）观察变化

**可能发现**：
- 大学及以上学历群体的生活满意度是否高于中学学历群体？
- 教育投资是否带来长期幸福感提升？
- 不同年龄段，教育对幸福感的影响是否发生变化？

**示例对比**：
- Andrew（剑桥大学法律）vs Tony（中学学历出租车司机）
- Nick（牛津大学物理，后成为教授）vs Jackie（中学毕业，家庭照护）

### 2. 个人一生中的幸福度变化

**问题**：一个人的人生轨迹中，幸福度如何随时间变化？

**探索方法**：
- 点击人物卡片查看成长轨迹
- 观察生活满意度折线图的波动
- 结合婚姻状况、职业变化等事件分析

**典型案例**：
- **Neil Hughes**：从7岁的活泼儿童（满意度4分），到21岁大学辍学、28岁流浪（满意度1分），再到42岁成为地方议员（满意度3分），呈现"U型"人生轨迹
- **Tony Walker**：从7岁想成为骑师（满意度4分），到35岁尝试演员失败（满意度3分），再到49岁家庭幸福（满意度4分），展现工人阶级的韧性
- **Suzy Lusk**：21岁时对婚姻持怀疑态度（满意度2分），28岁结婚后满意度提升至4分，体现家庭对幸福感的影响

### 3. 出身阶层与生活幸福度的关系

**问题**：出身贫穷和出身富裕的人，生活幸福度有何差异？

**探索方法**：
- 在对比分析中选择"成长背景"作为分组条件
- 选择"平均生活满意度"作为对比指标
- 观察不同年龄节点的阶层差异

**可能发现**：
- **上层/精英教育群体**（Andrew、John、Suzy）：
  - 教育资源丰富，职业路径清晰
  - 生活满意度普遍较高（3.5-4分）
  - 但也面临婚姻变故（如Andrew 56岁离婚）

- **工人阶级群体**（Tony、Jackie、Lynn、Sue）：
  - 教育机会有限，职业选择较少
  - 生活满意度波动较大（2-4分）
  - Jackie因残疾和伴侣去世，满意度降至2分
  - Tony通过努力工作和家庭支持，满意度保持较高

- **福利院/特殊成长背景**（Paul、Symon）：
  - 童年缺乏稳定家庭环境
  - 成年后生活满意度中等（3分左右）
  - Symon 42岁因与子女关系疏远，满意度降至3分

**数据洞察**：
- 阶层优势在年轻时更明显（21岁、28岁）
- 中年阶段（35-49岁），各阶层满意度差距缩小
- 晚年（63岁）时，家庭关系和健康成为更重要的幸福因素

### 4. 婚姻状况与幸福感

**问题**：婚姻状况如何影响生活满意度？

**探索方法**：
- 选择"婚姻状况"作为分组条件
- 对比"已婚"、"离异"、"单身"群体的满意度

**可能发现**：
- 已婚群体平均满意度较高
- 离异后满意度下降，但再婚可恢复
- 单身群体的满意度差异较大（Neil长期单身且满意度较低，John单身但满意度较高）

### 5. 职业与幸福感的关系

**问题**：不同职业类别的人群，生活满意度有何差异？

**探索方法**：
- 选择"工作类别"作为分组条件
- 对比"学者"、"教师"、"司机"、"家庭照护"等群体

**可能发现**：
- 学术界（Nick、Bruce）满意度中等偏上
- 法律界（Andrew、John）满意度较高
- 家庭照护者（Jackie、Lynn）满意度较低
- 退休后满意度普遍提升

### 6. 子女数量与生活满意度

**问题**：子女数量是否影响幸福感？

**探索方法**：
- 查看人物轨迹中的子女数量变化
- 对比不同子女数量群体的满意度

**可能发现**：
- 子女数量与满意度无直接线性关系
- 家庭支持系统比子女数量更重要
- 单亲抚养多个子女（Jackie）满意度较低

## 数据结构

### 人物基础信息

```javascript
{
  id: "tony",                    // 唯一标识
  name: "Tony Walker",           // 英文名
  cnName: "托尼·沃克",           // 中文名
  gender: "male",                // 性别
  originClass: "working",        // 成长背景
  birthYear: 1957,               // 出生年份
  status: "living",              // 当前状态
  summary: "...",                // 人物简介
  tags: ["east-london", "taxi-driver", "jockey"]
}
```

### 年龄节点数据

每个年龄节点（7、14、21、28、35、42、49、56、63岁）包含：

```javascript
{
  age: 35,                       // 年龄
  year: 1992,                    // 年份
  appeared: true,                // 是否参与拍摄
  residence: "伦敦东区",          // 居住地
  education: "secondary",        // 教育程度
  educationText: "中学",          // 教育程度（中文）
  maritalStatus: "married",       // 婚姻状况
  childrenCount: 2,              // 子女数量
  jobCategory: "driver",         // 工作类别
  jobText: "出租车司机",          // 工作类别（中文）
  lifeSatisfaction: "medium",    // 生活满意度（high/medium/low）
  lifeSatisfactionScore: 3,      // 生活满意度评分（0-5）
  notes: "..."                   // 备注
}
```

### 标准化字段取值

#### 成长背景（originClass）
- `upper`：上层/精英教育
- `middle`：中产
- `working`：工人阶级
- `care`：福利院成长
- `rural`：乡村

#### 婚姻状况（maritalStatus）
- `single`：单身
- `partnered`：伴侣关系
- `married`：已婚
- `divorced`：离异
- `remarried`：再婚
- `widowed`：丧偶
- `unknown`：未知

#### 教育程度（education）
- `primary`：小学
- `secondary`：中学
- `grammar`：文法学校
- `boarding`：寄宿学校
- `university`：大学
- `postgraduate`：研究生
- `none`：无
- `unknown`：未知

#### 工作类别（jobCategory）
- `student`：学生
- `academic`：学者
- `law`：法律
- `driver`：出租车司机
- `librarian`：图书馆员
- `administrator`：行政
- `teacher`：教师
- `counselor`：咨询/照护
- `musician`：音乐人
- `civil_service`：公务/公共部门
- `trades`：技工
- `homemaker`：家庭照护
- `politics`：地方政治
- `jockey`：骑师/赛马
- `retired`：退休
- `unknown`：未知

#### 生活满意度（lifeSatisfaction）
- `high`：高（评分4-5）
- `medium`：中（评分3）
- `low`：低（评分1-2）
- `unknown`：未知（评分0）

## 技术栈

- **前端框架**：原生JavaScript（无框架依赖）
- **数据可视化**：Canvas API（自实现图表）
- **样式**：CSS3（响应式设计）
- **数据存储**：静态JSON（内嵌于app.js）

## 部署说明

### 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/your-username/seven-up-cards.git
cd seven-up-cards
```

2. 启动本地服务器（任选其一）：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve

# 使用PHP
php -S localhost:8000
```

3. 访问 `http://localhost:8000`



### 图片资源处理

当前图片使用CBS News图集外链，如需本地化：

1. **图片格式**：转换为WebP格式（压缩率更高）
2. **存储路径**：`public/images/people/{personId}/{age}.webp`
3. **文件大小**：
   - 缩略图：30-60KB
   - 详情图：100-180KB
4. **懒加载**：使用Intersection Observer API实现
